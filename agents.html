🧠 Мультиагентная AI-система

CEO → AI-Директор → субагенты. Как я построил свою AI-команду
🎯 Управляю бизнесом через Telegram
💡 Что это и зачем

Представь, что у тебя есть команда ассистентов. Каждый — эксперт в своей области. WB-аналитик, инвестиционный советник, фитнес-наставник, персональный консультант по обучению.

Они не просят зарплату, не уходят в отпуск и не нуждаются в 3-месячном онбординге. Они работают 24/7, у каждого — своя память и специализация. И ты управляешь ими через один Telegram-чат.

Я построил такую систему для себя. Теперь я CEO, мой основной AI (Директор) — операционный директор, а субагенты — мои профильные помощники.

🔧 Как это работает технически

Основная инновация — я построил это на OpenClaw, open-source платформе для AI-агентов:

  • 🧠 Изолированная память — у каждого агента своя папка brains/, как отдельный мозг. Когда агент не работает, его память не смешивается с другими.
  • 🏷️ Переключение по тегам — пишешь #wb или #invest в чат, и контекст переключается на нужного агента. Без меню, без кнопок, без интерфейса — просто текст.
  • 💬 Всё в одном чате — все диалоги с любым агентом в одном Telegram-чате. CEO пишет задачу — Директор распределяет.
  • ⚡ Голосовой ввод — DeepSeek + Whisper: сказал — AI понял — агент ответил. Три секунды.
  • 🛠 Расширение без кода — нужен новый агент? Создаю папку brains/new_agent, настраиваю — готово. Ноль программирования.
🎯 Ключевые особенности

🧠 Раздельная память

Агенты не путают контекст. WB-аналитик не знает про фитнес, инвест-советник — про отчёты WB.

🏷️ Управление тегами

Простой текст — CEO. Тег — субагент. Никаких сложных интерфейсов.

⚡ Мгновенное переключение

Написал #wb — через секунду отвечает WB-аналитик. Написал без тега — отвечаю я как COO.

🎤 Голосовой ввод

Голосовое сообщение → Whisper транскрибация → DeepSeek анализ → ответ. 3 секунды на цикл.

📂 Файловая память

Каждый агент хранит память в текстовых файлах. Простая структура, никаких сложных БД для памяти.

🏗 OpenClaw + DeepSeek

Бесплатная open-source платформа + DeepSeek API. Минимум затрат, максимум возможностей.
